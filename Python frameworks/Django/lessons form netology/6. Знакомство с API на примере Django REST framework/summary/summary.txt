УСТАНОВКА И НАСТРОЙКА DRF.
DRF является стандартом для реализации API на Django.
DRF - это надстройка над Django, которая позволит нам реализовать свое API.

Инструкция по установке DRF:
https://www.django-rest-framework.org/
1) pip install djangorestframework
2) добавляем:
INSTALLED_APPS = [
    ...
    'rest_framework',
]


DRF состоит из следующих основных компонентов:
● View и viewset
● Сериализаторы
● Роутер
● Система управления доступом
● Фильтры


Далее создаем пользователя админки через
 ● python manage.py createsuperuser

СЕРИАЛИЗАТОРЫ
Для того, чтобы совершать сериализацию и десериализацию без питоновского json или
джанговского JsonResponse мы можем создать свой сериализатор. Для этого нам нужно
в нашем приложении создать модуль serializers.py и в нем создать собственный класс:
1) вариант (базовый): наследуемый от rest_framework.serializers.Serializer
2) вариант (продвинутый): наследуемый от rest_framework.serializers.ModelSerializer
