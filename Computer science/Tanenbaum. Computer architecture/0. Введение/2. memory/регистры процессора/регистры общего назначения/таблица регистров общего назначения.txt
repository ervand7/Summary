Таблица регистров общего назначения для архитектуры x86 (32-битная версия):

| Регистр           | Акроним        | Описание                                                      |
|------------------|---------------|--------------------------------------------------------------|
| EAX (R0)         | Accumulator   | Регистр аккумулятора (часто используется для результатов операций) |
| EBX (R1)         | Base          | Базовый регистр (часто используется для указателей на данные и адресов) |
| ECX (R2)         | Counter       | Регистр счетчика (часто используется в циклах)                   |
| EDX (R3)         | Data          | Регистр данных (используется для хранения данных и результатов операций) |
| ESI (R4)         | Source Index  | Индекс источника (часто используется для указателей на исходные данные) |
| EDI (R5)         | Destination Index | Индекс назначения (часто используется для указателей на целевые данные) |
| ESP (R6)         | Stack Pointer | Указатель стека (используется для управления стеком и передачей параметров функций) |
| EBP (R7)         | Base Pointer  | Указатель базы (часто используется для доступа к параметрам и локальным переменным в функциях) |

Это 32-битные регистры общего назначения, используемые в архитектуре x86. В режиме 64 бит,
есть расширенные версии этих регистров (например, RAX, RBX, RCX и так далее), и они
используются в 64-битных процессорах для работы с 64-битными данными и адресами.
