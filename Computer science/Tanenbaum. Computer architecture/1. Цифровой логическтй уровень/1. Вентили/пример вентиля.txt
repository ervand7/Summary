### Подробная схема простейшего вентиля (NOT) на nMOS:

                            +V (Питание, например, +5В)
                               │
                               │
                          ┌────┴─────┐
                          │  Резистор│ ← Ограничивает ток (Pull-up)
                          │   (R)    │
                          └────┬─────┘
                               │
                          (Сток/Drain)
                               │
                               │
                         ┌─────┴────────┐
                         │              │
Вход (A) ───────────────▶│ Затвор (Gate)│ ← Управляет проводимостью
                         │   Транзистор │ ← Это nMOS (n-type MOSFET)
                         │    (nMOS)    │ ← Это и есть основа логического ВЕНТИЛЯ
                         └─────┬────────┘
                               │
                          (Исток/Source)
                               │
                              GND (Земля)

                               ↑
                          Выход (Y)
               (точка между резистором и стоком)


✅ Где что:

| Элемент               | Что это и зачем?                                                  |
| --------------------- | ----------------------------------------------------------------- |
| **Резистор (R)**      | Подтягивает выход вверх, когда транзистор закрыт (лог. 1)         |
| **Транзистор (nMOS)** | Сам переключатель — именно он включает/выключает ток              |
| **Затвор (Gate)**     | Получает входной сигнал (A) — открывает или закрывает канал       |
| **Сток (Drain)**      | Подключается к резистору и выходу                                 |
| **Исток (Source)**    | Соединён с землёй (GND)                                           |
| **Выход (Y)**         | Получает лог. 0 или лог. 1 в зависимости от состояния транзистора |


👉 Это простейшая реализация **NOT-вентиля (инвертора)**:

* Если **Вход (A) = 0** → транзистор закрыт → **Выход (Y) = 1**.
* Если **Вход (A) = 1** → транзистор открыт → **Выход (Y) = 0**.


✅ Здесь нет **диодов**, потому что для цифровых вентилей на MOSFET они не нужны.
✅ Всё реализовано через **транзистор + резистор**.
