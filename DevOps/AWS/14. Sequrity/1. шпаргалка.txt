1. **AWS KMS (Key Management Service)** — сервис для управления криптографическими
ключами, используемыми для шифрования данных. Поддерживает создание, управление
и использование ключей шифрования.

2. **AWS CloudTrail** — сервис для мониторинга и логирования активности аккаунта
AWS. Записывает действия, выполняемые пользователями, ролями и сервисами, и
предоставляет историю событий для анализа и аудита.

3. **AWS Config** — сервис для оценки, аудита и мониторинга конфигураций
ресурсов AWS. Обеспечивает видимость изменений конфигураций и их соответствие
установленным требованиям.

4. **AWS Shield** — управляемый сервис защиты от DDoS атак. Shield Standard
предоставляется бесплатно и защищает ресурсы AWS от большинства распространенных
атак. Shield Advanced предлагает дополнительную защиту и поддержку.

5. **AWS WAF (Web Application Firewall)** — веб-аппликационный фаервол,
который защищает веб-приложения от распространенных веб-угроз, таких как
SQL-инъекции и скриптинг межсайтовых запросов (XSS).
Работает на уровне 7.

6. **Amazon Detective** — управляемый сервис, который помогает анализировать,
расследовать и быстро выявлять коренные причины потенциальных проблем
безопасности. Он автоматически собирает и обрабатывает данные из ваших AWS
ресурсов, такие как журналы CloudTrail, и создает граф связей для анализа
аномальной активности.

7. **AWS Inspector** — автоматизированный сервис для оценки безопасности
приложений, развернутых на инстансах EC2. Проверяет уязвимости и соблюдение
лучших практик безопасности.

8. **AWS Macie** — сервис для автоматического обнаружения, классификации
и защиты чувствительных данных, таких как персональные данные (PII) и
интеллектуальная собственность.

9. **AWS GuardDuty** — управляемый сервис для обнаружения угроз, который
постоянно мониторит ваши учетные записи и данные на предмет вредоносной
активности и несанкционированного поведения.

10. **AWS Secrets Manager** — сервис для управления секретами, такими как
учетные данные базы данных, API ключи и другие чувствительные данные.
Поддерживает автоматическое обновление и ротацию секретов.

11. **AWS Certificate Manager (ACM)** — сервис для управления SSL/TLS
сертификатами. Обеспечивает простое развертывание, управление и ротацию
сертификатов для использования с ресурсами AWS и собственными доменами.

12. **Encryption in Transit (шифрование при передаче данных)** — метод защиты
данных при их передаче между клиентами и серверами или между сервисами.

13. **Encryption at Rest (шифрование данных в состоянии покоя)** — метод
защиты данных, когда они хранятся на дисках, в базах данных или других хранилищах.

14. **AWS Security Hub** — централизованная панель для управления безопасностью,
которая собирает, анализирует и визуализирует данные безопасности из различных
сервисов AWS и партнерских решений.
