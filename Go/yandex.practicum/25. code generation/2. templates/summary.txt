А пакет html/template предназначен для генерации HTML. Он делает примерно то
же самое, что и text/template, а заодно предотвращает внедрение в шаблоны
вредоносного JavaScript-кода. В большинстве случаев html/template используется
в связке с веб-сервером и возвращает динамически сформированные HTML-страницы.
Шаблон состоит из управляющих конструкций в двойных фигурных скобках {{ }}:
{{pipeline}} — содержит строковое представление выражения (pipeline). Выражения
могут быть произвольными, к примеру:

 - {{.}} — текущее значение;
 - {{.Field}} — значение поля Field;
 - {{urlquery .Field}} — закодированное для URL значение поля Field;
 - {{call .SomeFunc 1 2}} — значение вызова функции SomeFunc с параметрами 1 2;
 - {{.Field | urlquery | call .SomeFunc}} — в сумме можно объединять последовательность
операций через символ |.
 - {{if pipeline}} T {{end}} — выводит содержимое T, когда указанное значение
pipeline не равно 0, пустой строке, false или имеет элементы, если pipeline — это слайс или мапа.
 - {{range pipeline}} T {{end}} — выводит содержимое T для каждого элемента pipeline,
в этом случае pipeline должен быть слайсом или мапой. pipeline внутри range будет равен текущему элементу.
