* Cache miss
  Ключа нет → 1000 запросов одновременно идут в БД

* Synchronized expiration
  TTL истёк у популярного ключа → весь трафик в БД в одну секунду

* Cache stampede
  Кэш пуст → many concurrent requests miss the cache and all hit the database for the same data

> Instead of absorbing load, cache failures can funnel traffic straight into the database.
