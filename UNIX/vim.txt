 ● vim
запуск и автоматическое попадание в командный режим

 ● i
попадаем в режим ввода текста

 ● esc
возвращение в командный режим
ВНИМАНИЕ! Не забываем эту команду, иначе не сможем выйти из vim

 ● :w test.txt
сохраняем документ как "test.txt"

 ● :q enter
выйти из vim с сохранением изменений

 ● vim test.txt
открываем созданный ранее файл "test.txt"

 ● :q! enter
выйти из vim без сохранения изменений

 ● :wq enter
сохранить и выйти из vim одной командой

 ● fn <- ->
перейти в начало/конец строки (в режиме ввода)

 ● e/b
шаг по одному слову вперед/назад (в командном режиме)

 ● /ура
поиск слова "ура" (в командном режиме)

 ● n
найти следующее вхождение слова "ура"

 ● v
переход в визуальный режим. Далее стрелочками выделяем нужное и удаляем через <x>

 ● u
отмена удаления
