1. **AWS Snowball** — устройство для физического переноса больших объемов данных в и из
AWS. Используется для безопасной передачи данных, когда передача по сети занимает
слишком много времени или стоит слишком дорого.

2. **AWS Snowball Edge** — усовершенствованная версия Snowball, которая не только
позволяет переносить данные, но и поддерживает локальные вычисления и хранение данных.
Имеет встроенные вычислительные ресурсы и хранение.

3. **AWS Snowmobile** — большой контейнер для физического переноса огромных объемов
данных (до 100 петабайт) в AWS. Представляет собой грузовик, который перевозит данные
на специальных контейнерах.

4. **Data Migration** (миграция данных) — процесс перемещения данных из одного места в
другое, например, из локального центра обработки данных в облако AWS, с использованием
устройств Snowball или Snowmobile.

5. **Edge Computing** (периферийные вычисления) — вычислительная модель, при которой
данные обрабатываются и анализируются ближе к источнику данных, а не в централизированном
облаке. Snowball Edge поддерживает периферийные вычисления.

6. **Encryption (шифрование)** — все данные, передаваемые с помощью Snowball и
Snowball Edge, шифруются для обеспечения безопасности. Ключи шифрования управляются с
помощью AWS Key Management Service (KMS).

7. **Import/Export (импорт/экспорт)** — процессы загрузки данных в AWS (импорт) или выгрузки
данных из AWS (экспорт) с использованием устройств Snowball, Snowball Edge или Snowmobile.
Ты загружаешь данные на устройство, отправляешь его в AWS, и они загружают данные в облако.
Для обратного процесса AWS копирует данные на устройство и отправляет его тебе.

8. **S3 Compatible Storage** (совместимое хранилище S3) — Snowball Edge предоставляет
интерфейс для хранения данных, совместимый с Amazon S3, что позволяет легко интегрировать
его с существующими приложениями и инструментами.

9. **Job Management** (управление заданиями) — процесс управления заданиями по переносу
данных с использованием консоли управления AWS или API. Включает создание, отслеживание и
завершение заданий.

10. **E Ink Display** — электронная чернильная дисплейная панель на устройствах Snowball и
Snowball Edge, которая используется для отображения информации о задании и упрощения логистики.

11. **Lambda@Edge** — функция AWS Lambda, работающая на устройствах Snowball Edge,
позволяющая выполнять код для обработки данных на самом устройстве, обеспечивая низкую задержку.

12. **Workflow** (рабочий процесс) — последовательность шагов, выполняемых для успешной
миграции данных с использованием устройств Snowball или Snowball Edge, включая упаковку
данных, отправку устройства, передачу данных и подтверждение завершения задания.

13. **Logistics and Shipping** (логистика и доставка) — процесс отправки и получения
устройств Snowball. AWS управляет логистикой, обеспечивая быструю и безопасную доставку устройств.

14. **Durability** (долговечность) — устройства Snowball и Snowball Edge разработаны для
работы в тяжелых условиях и обеспечивают высокую надежность хранения данных во время
транспортировки.

15. **Offline Data Transfer** (оффлайн-передача данных) — процесс передачи данных без
использования сетевого подключения, с использованием физических устройств для перемещения
данных между источниками и AWS.
