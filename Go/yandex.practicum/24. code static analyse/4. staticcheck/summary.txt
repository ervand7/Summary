 ● go install honnef.co/go/tools/cmd/staticcheck@latest
 Эта команда установит последнюю версию программы в директорию $GOPATH/bin

staticcheck работает подобно команде go vet. Например, staticcheck . проверит
файлы в текущей директории, а staticcheck ./... проверит файлы во всех поддиректориях.

Конфигурация
Проверки анализатора staticcheck разбиты на группы и имеют индекс
вида SA5001, S1000, ST1023:
 - SA1??? — неправильное использование стандартных библиотек;
 - SA2??? — проблемы с многопоточностью;
 - SA3??? — проблемы с тестами;
 - SA4??? — бесполезный код;
 - SA5??? — ошибочный код;
 - SA6??? — проблемы с производительностью;
 - SA9??? — сомнительные конструкции кода, c высокой вероятностью ошибочные;
 - S1??? — упрощения кода;
 - ST1??? — стилистические погрешности;
 - QF1??? — проверки, которые использует gopls для автоматического рефакторинга.

Имена необходимых для проверки правил можно указать в параметре -checks.
Чтобы получить информацию об определённом правиле, укажите его имя
после флага explain:
 ● staticcheck -checks "SA5000,SA5001" .
 ● staticcheck -explain SA5000
