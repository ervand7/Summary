A load balancer distributes traffic using routing algorithms.

Common algorithms:
* Round robin — rotate requests evenly
* Weighted round robin — stronger instances get more traffic
* Least connections — choose the least busy instance
* Hashing — same key goes to the same instance
* Consistent hashing — minimizes remapping when instances change

Key insight:
* Even traffic distribution does not guarantee even load

Interview version:
Load balancers use algorithms like round robin, least connections, and hashing
to distribute traffic across instances efficiently.
