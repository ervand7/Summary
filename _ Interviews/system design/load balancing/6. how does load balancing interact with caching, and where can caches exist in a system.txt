### Where caches can exist in a system
1. Client-side cache

* Browser, mobile app
* Reduces requests entirely
* No server involvement

2. CDN / edge cache
* Caches static and public content
* Reduces load before traffic reaches services

3. Load balancer / proxy cache
* NGINX, Envoy
* Caches HTTP responses centrally

4. Per-instance (local) cache
* In-memory cache inside each service
* Fast but duplicated across instances

5. Shared cache
* Redis, Memcached
* Single logical cache for all instances
