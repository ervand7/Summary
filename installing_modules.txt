Все это нужно делать через ТЕРМИНАЛ.
https://docs.python.org/3/library/venv.html - ссылка на документацию
https://pypi.org/search/?q=http - ссылка на каталог библиотек для скачивания
_ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _ _

Открываем ТЕРМИНАЛ и вводим команды:
● mkdir 'абсолютный_путь_до_папки' - создаем папку

● cd 'абсолютный_путь_до_папки' - переходим в нее

● python3 -m venv .env - говорим терминалу, что с помощью модуля venv (который 
создает виртуальное окружение) создай новое виртуальное окружение в папке .env

● ls -la - так просматриваем список файлов. В одном из них и будет наш питон (копия)

● ls -l .env/bin/ - смотрим, что лежит в bin (после ввода этой команды 
мы увидим путь, где лежит питон)

● source .env/bin/activate - активируем виртуальное окружение

_________________________ Это делать необязательно ________________________
● python - далее включаем python в интерактивном режиме, проверяем, есть ли
там нужный нам реквест или нет. Потом выходим с помощью exit()
____________________________________________________________________________


●  мы сейчас находимся в папке .env. Тут мы скачиваем реквест командой 
pip install requests (ну или все, что угодно с сайта https://pypi.org/)


_________________________ Это делать необязательно ________________________
● python - заходим снова в питон
● import requests - уже можем спокойно импортировать
____________________________________________________________________________

● Далее, уже в PyCharm нужно выбрать правильный virtual environment. Заходим 
в Preferences - Project: 'название_текущего_проекта' - python interpreter - шестеренку
в правом верхнем углу - add - Existing environment и выбираем
Внимание! Если уже имеется библиотека одной версии и мы устанавливаем ту же 
библиотеку, но другой версии, то старая версия удалится и новая запишется.
Внимание! Возможно этот пункт с первого раза не получится сделать,
Так как environment'у нужно немного прогрузиться. Со второго раза все получится.

|||||||||||| Инструкция по групповой установке пакетов |||||||||||||||||||||
● сейчас мы находимся там же, где и остановились на прошлой инструкции

● pip freeze - смотрим список установленных библиотек

● pip freeze > requirements.txt - создание отдельной папки для одновременного 
скачивания множества библиотек. Далее эту папку нужно будет открывать в 
Finder, записывать туда нужные модули
Список должен выглядеть так:
vk7
youtube3==0.1.2 (мы прописываем '==' если хотим указать какую-то конкретную версию)

●  pip install -r requirements.txt   - начинаем групповую установку пакетов
