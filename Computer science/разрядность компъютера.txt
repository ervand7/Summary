**Разрядность компьютера — это количество битов, которое процессор может
обработать или передать за один такт.**


### Кратко:
* Это **«ширина» данных и адресов**, которую процессор может обработать за один раз.
* Измеряется в **битах**: **8, 16, 32, 64 бита**.


### Что определяет разрядность:
| Параметр                                  | Чем выше разрядность — тем…                                    |
| ----------------------------------------- | -------------------------------------------------------------- |
| **Размер обрабатываемых данных**          | Можно обрабатывать более «широкие» числа (например, 64-битные) |
| **Максимальный объём оперативной памяти** | Например, 32-битная система → максимум 4 ГБ RAM                |
| **Ширина регистров процессора**           | Внутренние операции быстрее и эффективнее                      |
| **Тип операционной системы**              | Windows 32-bit ≠ Windows 64-bit                                |


✅ Примеры:
* **8-разрядные компьютеры** — старые приставки, микроконтроллеры (например, Atari, Z80).
* **16-разрядные** — первые IBM PC.
* **32-разрядные** — большинство ПК до начала 2010-х годов.
* **64-разрядные** — современные ПК, ноутбуки, смартфоны.
