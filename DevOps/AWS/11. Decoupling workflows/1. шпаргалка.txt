1. **Amazon SQS (Simple Queue Service)** — полностью управляемый сервис очередей
сообщений, который позволяет обмениваться сообщениями между различными компонентами
распределенных приложений. Обеспечивает декомпозицию и масштабируемость приложений.
Deduplication messages в контексте SQS — это механизм, который предотвращает
повторную обработку одинаковых сообщений. Эта функция доступна в очередях типа
FIFO (First-In-First-Out) и помогает избежать дублирования данных.
SQS scales automatically

2. **Amazon SNS (Simple Notification Service)** — сервис для отправки уведомлений
и рассылки сообщений. Позволяет отправлять сообщения в различные конечные точки,
такие как HTTP/HTTPS, email, SMS, или другие очереди SQS. Поддерживает паттерн pub/sub
(публикация/подписка).

3. **Amazon MQ** — управляемый брокер сообщений для Apache ActiveMQ и RabbitMQ.
Позволяет мигрировать существующие приложения, использующие сообщения, в AWS без
изменения кода.

4. **AWS Step Functions** — сервис для координации компонентов распределенных
приложений и микросервисов с использованием визуальных рабочих процессов. Обеспечивает
управление состояниями и последовательностью выполнения задач.

5. **AWS Lambda** — безсерверный сервис для выполнения кода в ответ на события.
Позволяет запускать функции в ответ на изменения данных, запросы HTTP или другие
события, обеспечивая декомпозицию приложений.

6. **Amazon EventBridge** — сервис для маршрутизации событий от различных источников
(например, SaaS приложений, AWS сервисов) к целевым приложениям и сервисам. Позволяет
строить приложения с архитектурой, основанной на событиях.

7. **Amazon Kinesis Data Streams** — сервис для захвата, обработки и хранения
потоковых данных в реальном времени. Подходит для приложений, требующих быстрой
обработки и анализа данных.

8. **Amazon Kinesis Data Firehose** — полностью управляемый сервис для захвата,
трансформации и загрузки потоковых данных в хранилища данных, такие как Amazon S3,
Redshift или Elasticsearch (OpenSearch Service).

9. **Amazon Managed Streaming for Apache Kafka (MSK)** — полностью управляемый
сервис для создания и управления кластерами Apache Kafka. Позволяет обрабатывать и
анализировать потоковые данные с высокой производительностью и низкой задержкой.

10. **AWS Glue** — полностью управляемый сервис для подготовки и интеграции данных.
Поддерживает создание, выполнение и управление рабочими процессами для ETL (extract,
transform, load) задач.

11. **Amazon Simple Workflow Service (SWF)** — координационный сервис для управления
состояниями и последовательностью выполнения задач в распределенных приложениях.
Подходит для длительных рабочих процессов.

12. **AWS Fargate** — безсерверный сервис для развертывания и управления контейнерами.
Позволяет запускать контейнеры без управления серверами или кластером.

13. **Amazon ECS (Elastic Container Service)** — управляемый сервис оркестрации
контейнеров, который позволяет запускать, останавливать и управлять контейнеризированными
приложениями на кластерах EC2 или Fargate.

14. **Amazon EKS (Elastic Kubernetes Service)** — управляемый сервис для запуска
Kubernetes на AWS. Обеспечивает оркестрацию контейнеров с использованием Kubernetes.

15. **AWS App Mesh** — сервис для управления микросервисами с использованием сетевой
структуры (service mesh). Обеспечивает надежное и безопасное взаимодействие между микросервисами.

16. **AWS Batch** — управляемый сервис для выполнения пакетных вычислений.
Автоматически создает и управляет вычислительными ресурсами для выполнения задач.
