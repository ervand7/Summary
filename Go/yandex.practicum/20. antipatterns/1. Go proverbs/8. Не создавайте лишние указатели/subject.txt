Используйте указатели в параметрах функции только в крайнем случае.
Если функция не меняет параметр, передавайте этот аргумент по значению.
При передаче структуры Go быстро отводит память на стеке и на выполнение
уходит меньше времени, чем при использовании указателей.

type Opts struct{
    ParamA int
    ParamB int
    ParamC int
    ParamD string
}

/* плохо */
func Config(opts *Opts) {}

/* хорошо */
func Config(opts Opts) {}

Не используйте указатели для слайсов, мап и каналов, так как переменные
этих типов представляют собой указатели по умолчанию. Также не
используйте указатели для скалярных типов.

/* плохо */
var (
    name *string
    vals *map[string]string
)
func F(p *int, s *[]string) {}

/* хорошо */
var (
    name string
    vals map[string]string
)
func F(p int, s []string) {}
