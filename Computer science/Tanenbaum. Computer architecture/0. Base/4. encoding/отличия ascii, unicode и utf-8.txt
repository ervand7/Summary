Давайте рассмотрим примеры символов для каждой из этих кодировок:

**ASCII** (American Standard Code for Information Interchange):

ASCII ограничен и включает основные символы, такие как буквы, цифры и знаки
препинания. Вот несколько примеров символов в ASCII:

- A (буква A)
- 7 (цифра 7)
- $ (знак доллара)
- ! (восклицательный знак)

**Unicode**:

Unicode предоставляет кодовые точки для символов разных письменных систем.
Примеры символов Unicode:

- Я (кириллическая буква "Я")
- ƒ (латинская буква "ƒ")
- 日 (китайская иероглифика "日")
- ♥ (символ "сердце")

**UTF-8** (Unicode Transformation Format - 8-bit):

UTF-8 представляет символы Unicode в виде последовательности байтов
переменной длины. Примеры символов в UTF-8:

- Буква "A" кодируется как `41` в шестнадцатеричной системе (или как `0x41`),
что в двоичной системе будет `01000001`.
- Кириллическая буква "Я" кодируется как `D0 AF` в шестнадцатеричной системе
(или как `0xD0 0xAF`), что в двоичной системе будет `11010000 10101111`.
- Символ "♥" кодируется как `E2 99 82` в шестнадцатеричной системе
(или как `0xE2 0x99 0x82`), что в двоичной системе будет `11100010 10011001 10000010`.

UTF-8 использует разное количество байтов для разных символов, что делает его
гибким и способным представлять символы из разных письменных систем в одной
кодировке.
