**Шина данных** — это физический и логический **канал для передачи данных** между
компонентами компьютера (например, между процессором, памятью, видеокартой).


### Зачем нужна шина данных:
✅ Чтобы **перемещать биты (данные)** между устройствами внутри компьютера:
* CPU ↔ RAM
* CPU ↔ видеокарта
* CPU ↔ устройства ввода-вывода


### Физически из чего состоит шина данных:

✅ Это:
* **Набор параллельных проводников (медных дорожек или соединений)** на материнской
плате или внутри микросхем.
* Эти проводники передают **электрические сигналы**.


### Как физически передаются данные:
1. Каждый бит — это просто **напряжение или его отсутствие** на проводнике.
   * Например:
     * 0 = 0 Вольт,
     * 1 = 1.2 Вольта (зависит от стандарта).

2. Если шина данных имеет, например, **64 провода (64-битная шина)**:
* За **один такт** можно передать **64 бита информации одновременно** (1 бит по каждому
проводнику).


### Пример:
CPU ──[64 параллельных дорожек]── RAM

* В каждый момент времени CPU может послать или получить 64 бита данных по этим проводникам.


### Как это возможно физически:
✅ За счёт:
* **Проводников** (металлических дорожек),
* **Электрических импульсов** (напряжений),
* **Тактового сигнала**, который синхронизирует отправку/получение.


✅ Итог в двух словах:
* **Шина данных** — это физический набор проводов, по которым одновременно передаются
биты в виде **электрических сигналов**.
* Это «дорога», по которой компьютер «разговаривает» с памятью и устройствами.
