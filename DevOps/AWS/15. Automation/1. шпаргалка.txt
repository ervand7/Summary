1. **AWS CloudFormation** — сервис для автоматизации развертывания инфраструктуры
как кода (Infrastructure as Code, IaC). Позволяет создавать, обновлять и удалять
AWS ресурсы с помощью шаблонов JSON или YAML. Основные компоненты: шаблоны (templates),
стэки (stacks).

2. **AWS Elastic Beanstalk** — платформа как услуга (PaaS) для развертывания и управления
веб-приложениями. Поддерживает автоматическое управление инфраструктурой, масштабирование
и мониторинг. Поддерживает различные языки программирования и контейнеры.

3. **AWS CodeDeploy** — сервис для автоматизации развертывания приложений на различные
вычислительные ресурсы, такие как EC2, Fargate, Lambda и on-premise сервера. Поддерживает
стратегии развертывания: in-place и blue/green.

4. **AWS CodePipeline** — сервис для автоматизации процессов непрерывной интеграции и
непрерывного развертывания (CI/CD). Позволяет создавать конвейеры для автоматизации этапов
сборки, тестирования и развертывания.

5. **AWS OpsWorks** — конфигурационный сервис, поддерживающий Chef и Puppet для управления
конфигурациями серверов. Подходит для управления динамическими масштабируемыми приложениями
и автоматизации развертывания приложений.

6. **AWS Systems Manager** — сервис для управления инфраструктурой и операциями.
Предоставляет возможности инвентаризации ресурсов, управления патчами, автоматизации задач
и выполнения команд на EC2 инстансах с помощью Run Command.

7. **AWS Lambda** — безсерверный сервис для выполнения кода в ответ на события. Позволяет
запускать функции в ответ на изменения данных, запросы HTTP или другие события.
Обеспечивает автоматическое масштабирование и оплату только за время выполнения.

8. **AWS Step Functions** — сервис для координации компонентов распределенных приложений
и микросервисов с использованием визуальных рабочих процессов. Обеспечивает управление
состояниями и последовательностью выполнения задач.

9. **Amazon EventBridge** — сервис для маршрутизации событий от различных источников
(например, SaaS приложений, AWS сервисов) к целевым приложениям и сервисам. Позволяет
строить приложения с архитектурой, основанной на событиях.

10. **AWS Glue** — полностью управляемый сервис для подготовки и интеграции данных.
Поддерживает создание, выполнение и управление рабочими процессами для задач ETL
(extract, transform, load).
