Vectorization means applying operations to whole arrays at once.

Instead of:
* Looping in Python

Do:
* Single operation on arrays

Why faster:
* Runs in optimized C code
* Fewer Python-level operations

Key idea:
> Vectorization replaces Python loops with native array ops


```
# slow (Python loop)
result = [x * 2 for x in range(5)]

# fast (vectorized)
import numpy as np
arr = np.arange(5)
result = arr * 2
```
