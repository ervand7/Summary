Архивирование:

 ● tar cvf my_tar.tar dir_1 dir_2
команда tar объединяет (архивирует) в один файл различные файлы директории
Эта команда НЕ сжимает, а только объединяет для удобства. Например, это
может быть полезно при переносе файлов
c - create
v -verbose (для подробного вывода информации о происходящем)
f - file (всегда должен идти последним аргументом)

 ● tar tf my_tar.tar
t - test
посмотреть что внутри архива

 ● tar xvf my_tar.tar
x - extract
распаковать архив

Компрессия:

gzip/bzip2/xz
эти команды компрессируют исключительно файлы

 ● gzip my_tar.tar
скомпрессируем my_tar.tar. Обратите внимание, что у my_tar.tar
появится новое расширение gz (файл станет my_tar.tar.gz)
 ● gunzip my_tar.tar.gz
раскомпрессируем

 ● bzip2 my_tar.tar
компрессирует эффективнее, чем gzip
 ● bunzip2 my_tar.tar.bz2
раскомпрессируем

 ● xz my_tar.tar
компрессируем
 ● unxz my_tar.tar
раскомпрессируем

 ● tar cvzf my_zip.gz dir_1 dir_2
z - zip
заархивировать и скомпрессировать директории (итоговое расширение будет .gz)
 ● tar tf my_zip.gz
смотрим содержимое заархивированного+скомпрессированного
 ● tar cvjf my_zip.gz

 ● zip –r myZIP.zip Folder1
запаковать Folder1 в ZIP
 ● unzip myZIP.zip
распаковать файл myZIP.zip
