 ● rm -rf <название папки>
удалить папку
 ● rm -R папка
рекурсивно удалить все содержимое папки
 ● rm <file.txt>
удалить файл

Создание папок/файлов
 ● mkdir my_folder
создание новой папки внутри текущей директории
 ● mkdir -p qwe/rty/asd
создать папку с подпапками

 ● touch file1.txt
создание нового файла внутри текущей директории
 ● touch {app, main, config, migrate}.py
создать за раз много файлов с расширением .py

 ● cp file.txt new_file.txt
копирование файла
 ● cp *.http /home/sammy
скопировать все файлы с расширением .http в директорию /home/sammy
 ● cp -R -v first/ second/
рекурсивное копирование директории. Флаг -v (verbose) визуализирует процесс

 ● mv <что переместить> <куда переместить>
переместить файл в определенную папку. Также этой командой можно переименовать

 ● > любой_файл.txt
создание нового файла внутри текущей директории.
После этой команды терминал ждет ввода данных, которые будет содержать
этот файл. После ввода данных, либо при нежелании ввода нужно нажать control+C
