src/runtime/string.go

type stringStruct struct {
    str unsafe.Pointer // указатель на байтовый массив (неизменяемый)
    len int            // длина строки в байтах
}

Строки в Go байтово-ориентированные: они хранят произвольную последовательность
байтов, а не обязательные Unicode-символы. Один символ в UTF-8 может занимать
от одного до четырёх байтов.

Строки оптимизированы под производительность. Они могут указывать на данные,
которые находятся внутри других объектов (например, массива байт или литерала).
Создание подстроки может выполняться как с копированием данных, так и без него.
В версиях Go до 1.20 подстроки почти всегда разделяли память с исходной строкой.
Начиная с Go 1.20 рантайм может копировать данные подстроки, чтобы избежать
удержания большого исходного массива в памяти.

Пример (до go1.20):
b := []byte("hello world")
s := string(b) // строка s указывает на те же байты, что и b (копирования нет)
