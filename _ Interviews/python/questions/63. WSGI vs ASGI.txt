👉 WSGI и ASGI — это контракты (как сервер должен вызывать Python-приложение),
а Gunicorn и Uvicorn — это серверы, которые реализуют эти контракты и реально
принимают HTTP и вызывают твой код.


WSGI (Web Server Gateway Interface):
* Synchronous interface
* One request per thread/process
* Designed for blocking I/O
* Used by Flask, Django (classic)

ASGI (Asynchronous Server Gateway Interface):
* Asynchronous interface
* Supports async/await
* Handles long-lived connections (WebSockets)
* Used by FastAPI, Starlette, modern Django

Key idea:
> WSGI = sync, ASGI = async + real-time
