## 2ï¸âƒ£ What is ASGI?

ASGI (Asynchronous Server Gateway Interface) is the modern successor to WSGI.

### Key properties

* Asynchronous (async/await)
* Event-loop based
* Supports multiple protocols
* Non-blocking I/O

### Supported protocols

* HTTP
* WebSockets
* Server-Sent Events (SSE)
* Background tasks

### Typical stack

```
Client â†’ Nginx â†’ Uvicorn/Hypercorn â†’ FastAPI/Django ASGI
```

### What ASGI is good at

* WebSockets & real-time apps
* Streaming responses
* High-concurrency APIs
* I/O-bound systems

## 3ï¸âƒ£ Core difference (one sentence)

> WSGI handles one request at a time per worker;
> ASGI handles many concurrent requests in one worker.
