 ● gRPC (Remote Procedure Calls) — это система удалённого вызова
 процедур (RPC) с открытым исходным кодом, первоначально разработанная в
 Google в 2015 году. В качестве транспорта используется HTTP/2.
 gRPC сериализует данные в бинарный формат Protocol Buffers.
 gRPC также называют протоколом.

 ● язык Protocol Buffers
 В файле со спецификацией API описываются
  - структуры сообщений (message)
  - службы (service)
 Файл должен иметь расширение .proto и начинаться с syntax = "proto3";
 Также нужно указать package <name>, что определяет имя пакета для
 сгенерированных файлов.
