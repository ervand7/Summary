**Битовая карта (Bitmap или битовое отображение)** - это структура данных, используемая
для представления бинарных данных или для отслеживания наличия и отсутствия элементов
в наборе данных. Она состоит из последовательности битов, где каждый бит представляет
определенное состояние или свойство элемента. Обычно битовая карта используется
для эффективного хранения информации о большом количестве элементов, где каждый
элемент может быть либо "включен", либо "выключен".

Примеры использования битовых карт:

1. **Хранение изображений**: В компьютерной графике, каждый пиксель изображения
может быть представлен с использованием битовой карты. Например, черно-белое изображение
может использовать 1 бит для каждого пикселя, где 0 представляет черный цвет, а 1 - белый.
Цветное изображение может использовать более битов для каждого пикселя.

2. **Управление файловой системой**: Файловые системы могут использовать битовые карты для
отслеживания использования блоков на диске. Каждый бит может представлять блок данных: 0,
если блок свободен, и 1, если блок занят.

3. **Отслеживание состояния ресурсов**: Битовые карты могут использоваться для отслеживания
состояния ресурсов, таких как память, порты ввода/вывода и другие. Каждый бит может
представлять доступность или занятость ресурса.

4. **Индексирование**: Битовые карты могут использоваться для создания индексов и
отображения наличия или отсутствия элементов в больших наборах данных.

Пример битовой карты для хранения статуса блоков на диске:

```
01011001 00100000 11010000 11111111 00000100
```

В данном примере каждый бит представляет блок данных на диске. 1 обозначает занятый блок,
а 0 обозначает свободный блок. С использованием битовой карты, можно быстро определить,
какие блоки свободны и доступны для записи данных, а какие уже заняты.
