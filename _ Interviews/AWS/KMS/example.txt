1. Сервис (Lambda / app) отправляет данные в AWS KMS на шифрование.
2. KMS шифрует данные ключом и возвращает ciphertext.
3. Сервис сохраняет ciphertext (DB / S3 / Secrets Manager).
4. Когда нужно — сервис снова отправляет ciphertext в KMS.
5. KMS проверяет IAM-права и возвращает plaintext для бизнес-логики.
