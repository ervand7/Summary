1. **Amazon SageMaker** — полностью управляемый сервис для разработки, обучения
и развертывания моделей машинного обучения (ML). Поддерживает весь жизненный цикл ML,
включая подготовку данных, выбор моделей, обучение и развертывание.

2. **SageMaker Studio** — интегрированная среда разработки (IDE) для машинного обучения,
предоставляющая инструменты для подготовки данных, создания моделей, их обучения и
развертывания в одном месте.

3. **SageMaker Notebooks** — управляемые Jupyter ноутбуки для разработки и тестирования
моделей машинного обучения. Поддерживают масштабируемые вычислительные ресурсы.

4. **SageMaker Training Jobs** — сервис для запуска тренировочных заданий для моделей
машинного обучения на управляемых инфраструктурах с масштабируемыми ресурсами.

5. **SageMaker Endpoints** — полностью управляемые конечные точки для развертывания
моделей машинного обучения и предоставления предсказаний в реальном времени.

6. **SageMaker Ground Truth** — сервис для создания высококачественных наборов данных
для обучения моделей машинного обучения с использованием методов активного обучения и
человеческой разметки.

7. **SageMaker Autopilot** — инструмент для автоматизации процесса создания, обучения
и настройки моделей машинного обучения, который автоматически создает и сравнивает
несколько моделей для выбора наилучшей.

8. **SageMaker Debugger** — инструмент для анализа и отладки тренировочных заданий
машинного обучения, который помогает выявлять проблемы в моделях и улучшать их качество.

9. **SageMaker Model Monitor** — сервис для мониторинга производительности и качества
развернутых моделей машинного обучения, который помогает выявлять отклонения и изменения
в данных.

10. **SageMaker Neo** — инструмент для оптимизации моделей машинного обучения для
выполнения на различных аппаратных платформах, таких как процессоры, графические
процессоры и FPGA.

11. **Amazon Rekognition** — сервис для анализа изображений и видео, который предоставляет
функции распознавания объектов, лиц, текста и активности.

12. **Amazon Polly** — сервис для преобразования текста в естественную речь, который
поддерживает несколько языков и голосов.

13. **Amazon Translate** — сервис для машинного перевода текста между различными
языками.

14. **Amazon Transcribe** — сервис для автоматической транскрипции аудио в текст,
поддерживающий несколько языков и различных форматов аудио.

15. **Amazon Comprehend** — сервис для обработки естественного языка (NLP), который
анализирует текст и извлекает из него полезную информацию, такую как сущности,
ключевые фразы, тональность и язык.

16. **Amazon Lex** — сервис для создания разговорных интерфейсов и чат-ботов с
использованием технологии распознавания речи и обработки естественного языка.

17. **Amazon Forecast** — сервис для создания прогнозных моделей на основе временных
рядов данных, который помогает предсказывать будущее поведение и тенденции.

18. **Amazon Personalize** — сервис для создания персонализированных рекомендаций и
контента, который использует машинное обучение для анализа поведения пользователей и
предпочтений.

19. **Amazon Textract** — сервис для извлечения текста и данных из сканированных
документов и изображений с использованием технологии оптического распознавания
символов (OCR).

20. **AWS DeepLens** — камера с поддержкой глубокого обучения, которая позволяет
разработчикам создавать и развертывать модели машинного обучения для анализа видео
на самом устройстве.

21. **AWS DeepRacer** — платформа для обучения и соревнований по автономным гоночным
автомобилям на основе машинного обучения с использованием обучения с подкреплением.

22. **AWS Glue** — полностью управляемый сервис для подготовки и интеграции данных.
Поддерживает создание, выполнение и управление рабочими процессами для ETL (extract,
transform, load) задач, что важно для подготовки данных для моделей машинного обучения.

23. **AWS Data Pipeline** — веб-сервис для автоматизации перемещения и преобразования
данных. Позволяет создавать сложные рабочие процессы для обработки данных.

24. **Amazon Kinesis** — платформа для обработки потоковых данных в реальном времени.
Поддерживает захват, обработку и анализ данных с высокой пропускной способностью и
низкой задержкой.

25. **AWS Lambda** — серверлесс-сервис для выполнения кода в ответ на события. Может
использоваться для предобработки данных или вызова моделей машинного обучения.
