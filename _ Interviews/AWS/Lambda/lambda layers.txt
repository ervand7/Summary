Lambda Layers are shared packages for Lambda functions.

They are used to:

* Share common libraries (e.g. Python deps)
* Share runtime code (utils, helpers)
* Reduce duplicate deployment size

Example:

* Layer contains `requests`, `boto3`, shared utils
* Multiple Lambdas reuse the same layer

* AWS stores and versions it
