Load balancing affects caching because it determines which instance receives requests.

Cache types:
* Client/browser cache
* CDN/edge cache
* Proxy cache (NGINX, Envoy)
* Local in-memory cache
* Shared cache (Redis, Memcached)

Key interaction:
* Random routing lowers local cache hit rates
* Sticky sessions improve local cache hits but hurt scalability
* Shared caches work best with load balancing

Key insight:
* Load balancing distributes traffic, while caching concentrates data

Interview version:
Load balancing affects cache hit rates and consistency; shared or edge caches usually
work best behind load balancers.
