Числа Фибоначчи — это последовательность чисел, в которой каждое число
является суммой двух предыдущих, а первые два числа обычно равны 0 и 1.
Таким образом, последовательность начинается так:
0, 1, 1, 2, 3, 5, 8, 13, 21, и так далее.

Формально, числа Фибоначчи \(F(n)\) определяются следующим образом:

\[ F(0) = 0, \quad F(1) = 1 \]

\[ F(n) = F(n-1) + F(n-2) \text{ для } n > 1 \]

Таким образом, каждое число Фибоначчи равно сумме двух предыдущих
чисел в последовательности. Эта последовательность названа в честь
итальянского математика Леонардо Пизанского, известного как Фибоначчи.
Числа Фибоначчи возникают в различных областях математики, информатики
и прикладных наук.
